# TaskManager
## Gerenciar tarefas cotidianas a partir de um arquivo .csv

### config.properties 
- medida=SEGUNDOS
- intervalo=1
- arquivo=tarefas.csv
  
### arquivo (definido em config.properties)
- nome,descricao,local,dataInicio,dataTermino
  
### formato das datas: 
- dd/MM/yyyy HH:mm ou dd/MM/yyyy
